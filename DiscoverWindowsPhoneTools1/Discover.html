Windows Phone SDK 8.0

Mise en route de la création d'applications pour Windows Phone 8 ou Windows Phone 7.5 avec Windows Phone SDK 8.0.

Windows Phone SDK 8.0 étend Visual Studio 2012 afin de vous fournir les outils dont vous avez besoin pour développer des applications pour Windows Phone, notamment des applications Windows Phone et des modèles de bibliothèque, l'émulateur Windows Phone Emulator ainsi que les outils d'analyse des performances. Vous pouvez utiliser cette extension Visual Studio pour compiler et déboguer vos applications Windows Phone, puis les déployer sur un Windows Phone ou Windows Phone Emulator.

Remarque    Le projet dans lequel apparaît cette page comporte des instructions relatives à l'installation de Windows Phone SDK 8.0, et non au kit de développement logiciel (SDK) proprement dit. Pour télécharger le kit de développement logiciel (SDK), cliquez sur le bouton the Télécharger Windows Phone SDK 8.0 ou accédez à la page [download page URL].
